-----------------------------------
Ce projet consiste a un site web pour numiriser la location des terrains de foot.


Acteurs:
-Utilisateur (Joueur/Societe)
-Gerant d'un complexe
-Admin

-----------------------------------
Configuration minimale:

-XAMPP 1.8.1 (ou ulterieur)
[Apache 2.4.3 /MySQL 5.5.27 PHP 5.4.7] 
WAMPP peut causer des problemes dans quelques pages.

-----------------------------------
HOW TO?

- Copie le dossier Tunifoot dans le dossier htdocs sous xampp(ou www sous wampp)
- Ouvre phpmyadmin et crer une base de donne sous le nom "tunifoot" (sans les " ")
- Importe le fichier tunifoot.sql


Des comptes d'utilisateurs sont deja predefinis, mais vous pouvez creer le votre.

-----------------------------------

Le siste est base sur une Fork de Bootstrap - Twitter.
L'espace Admin (backend) est base sur "Charisma" qui est base sur Bootstrap - Twitter.

Plusieurs bibliotheques et API sont implementes,si une Google API ne marche plus, veilliez changer les parametres de celles de votre compte il ce peut que ce compte a ete desactive.
